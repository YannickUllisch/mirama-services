# Project Management Microservice

A minimal, modular microservice for managing Miramas projects and related entities, built with ASP.NET Core, Clean Architecture with CQRS and Domain-Driven Design principles.

## Features

- Project and team management
- JWT authentication (for zero-trust)
- API versioning
- Clean separation of concerns (Domain, Application, Infrastructure, API)
- Ready for containerization and cloud deployment
